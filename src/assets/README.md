# Assets Folder Documentation

## Purpose
The `src/assets` folder stores static assets that are imported directly in React source files.

## Current Files
- `react.svg`: SVG asset available for source-level imports.

## Usage Guidance
- Add images, SVGs, and media here when they are imported in JavaScript/JSX.
- Prefer optimized formats and appropriately sized files.
- Use descriptive filenames to make asset intent clear.
