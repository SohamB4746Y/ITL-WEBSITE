# Components Folder Documentation

## Purpose
The `src/components` folder contains all reusable and page-section React components used by the ITL website.

## Component Inventory
- `Navbar.jsx`: Sticky top navigation with responsive mobile overlay menu.
- `HeroSection.jsx`: Landing hero with primary messaging and animated visual accents.
- `NorthStarSection.jsx`: Vision section presenting the organization’s north star.
- `FounderSection.jsx`: Founder introduction and profile call-to-action.
- `WhyWeExistSection.jsx`: Mission section with three strategic pillar cards.
- `TransformSection.jsx`: Methodology section with cascading capability cards.
- `PathwaysSection.jsx`: Four-pathway ecosystem section for different audiences.
- `Footer.jsx`: Footer with navigation shortcuts, social links, and brand summary.

## Conventions
- Component files use PascalCase naming.
- Each component exports one default React function component.
- Animation behavior is implemented through `framer-motion`.
- Iconography is provided through `lucide-react`.
